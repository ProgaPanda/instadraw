# instadraw
a live drawing app using sockets.io


## Installation

Use the package manager [npm](https://nodejs.org/) to install dependencies.
```bash
npm install
```
## Running locally
```bash
node server.js
```

## Online version
hosted on heroku: [instadraw](https://socketshare.herokuapp.com)
